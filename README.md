# Семейный бюджет 💰

Веб-приложение для планирования семейного бюджета: обязательные траты жены и
мужа, расчёт распределения дохода (с учётом годовой премии), резервного буфера и
свободных средств. Вверху — сводка по всей семье.

## Что умеет

- 📊 Наглядная диаграмма распределения дохода по категориям
- ✏️ Все значения редактируются прямо в интерфейсе (доход, статьи, суммы)
- 🛟 Резервный буфер — настраиваемый процент от дохода (по умолчанию 10%)
- ➕ Добавление и удаление статей расходов
- 💾 Автосохранение в браузере (localStorage)
- 📱 Адаптивная вёрстка для телефона и десктопа

> Буфер считается как процент от дохода (по умолчанию 10%). Процент можно менять

> месяц (премия ÷ 12) и добавляется к доходу.


## Авторизация (защита паролем)

Защита включается штатными средствами Vercel — **Deployment Protection**,
без кода в репозитории. В настройках проекта на Vercel:

**Settings → Deployment Protection**

- **Vercel Authentication** — доступ только тем, кто вошёл в Vercel и имеет
  права на проект (бесплатно на всех тарифах).
- **Password Protection** — единый общий пароль для всех, кому дашь ссылку
  (платная опция Vercel).

Выбери нужный режим, сохрани — защита применится к сайту.

## Стек

Vite + React + TypeScript, без тяжёлых зависимостей (диаграмма нарисована на SVG).

## Что дальше

Модель данных (`src/types.ts`) рассчитана на несколько участников, поэтому легко
добавить новые блоки и категории. Сводка по семье вверху появляется
автоматически, когда участников больше одного.

# Семейный бюджет 💰

Веб-приложение для планирования семейного бюджета. На старте — обязательные
траты жены: расчёт распределения дохода, резервного буфера и свободных средств.

## Что умеет

- 📊 Наглядная диаграмма распределения дохода по категориям
- ✏️ Все значения редактируются прямо в интерфейсе (доход, статьи, суммы)
- 🛟 Резервный буфер — настраиваемый процент от дохода (по умолчанию 10%)
- ➕ Добавление и удаление статей расходов
- 💾 Автосохранение в браузере (localStorage)
- 📱 Адаптивная вёрстка для телефона и десктопа

## Стартовые данные (жена)

| Статья           | Сумма   |
| ---------------- | ------- |
| Квартира         | 45 000 ₽ |
| Путешествия      | 30 000 ₽ |
| Еда              | 25 000 ₽ |
| Медицина         | 7 000 ₽  |
| Подписки         | 2 000 ₽  |
| Мелкие расходы   | 10 000 ₽ |
| Одежда           | 7 000 ₽  |
| Косметика        | 2 000 ₽  |
| **Итого трат**   | **128 000 ₽** |
| Буфер (10% от ЗП) | 27 000 ₽ |
| **Зарплата**     | **270 000 ₽** |
| **Свободно**     | **115 000 ₽** |

> Буфер считается как процент от дохода (по умолчанию 10%). Процент можно менять
> ползунком — расчёт обновится сразу.

## Запуск

```bash
npm install
npm run dev      # режим разработки
npm run build    # production-сборка в dist/
npm run preview  # предпросмотр сборки
```

## Стек

Vite + React + TypeScript, без тяжёлых зависимостей (диаграмма нарисована на SVG).

## Что дальше

Модель данных (`src/types.ts`) рассчитана на несколько участников — можно
добавить блок мужа и считать общий семейный бюджет: при появлении второго
участника вверху автоматически появится сводка по семье.
